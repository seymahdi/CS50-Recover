# CS50 Recover

This repository contains my personal solution to the Recover problem from Harvard University's CS50x course.

## Description

The Recover project focuses on digital forensics by recovering JPEG images from a raw forensic memory card image. It demonstrates how file signatures can be used to identify and reconstruct image files.

## Language

- C

## Concepts Practiced

- File I/O
- Digital Forensics
- JPEG File Signatures
- Memory Management
- Binary Data Processing

## Features

- Reads a raw memory card image
- Detects JPEG file signatures
- Recovers JPEG images
- Writes recovered images to separate files
